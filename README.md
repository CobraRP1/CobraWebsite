# CobraWebsite

Website for the Mannheim Center for Corporate Behavior and Regulation Analysis (Cobra).

Astro static site; all content lives in `src/data/*.yaml`. Push to `main` deploys to GitHub
Pages: **https://cobrarp1.github.io/CobraWebsite/**

```bash
npm install
npm run dev      # http://localhost:4321/CobraWebsite/
npm run build
```

- **[WORKFLOWS.md](WORKFLOWS.md)** — how to add a paper, a person, an event, news; how to add
  the logo, turn on analytics, move to a custom domain.
- **[CLAUDE.md](CLAUDE.md)** — architecture, content model, and the rules this repo is built on.

⚠️ Not ready to publicise yet: the design is a neutral placeholder (no corporate-design assets
yet), most content is empty, and the Impressum / Datenschutzerklärung are unreviewed drafts.
